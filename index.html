Step 1 — Supabase Table Banao

Apne Supabase project mein SQL Editor open karo aur ye query run karo:

CREATE TABLE IF NOT EXISTS projects ( id TEXT PRIMARY KEY, data JSONB NOT NULL, updated_at TIMESTAMPTZ DEFAULT NOW() ); -- Row Level Security disable karo (ya apni policy set karo) ALTER TABLE projects DISABLE ROW LEVEL SECURITY;

⚠ Production mein RLS enable karo aur proper policies set karo.
